All tuppers
GET /api/tuppers
Example: localhost:8080/api/tuppers
Add tupper
POST /api/tuppers/
Example: POST http://localhost:8080/api/tuppers/, Header: Content-Type: application/json Body: { "name": "Dinner", "description": "...", "times":{"begin":"2015-11-15T19:00:00.000Z","end":"2011-11-16T03:00:00.000Z"}, "location", { ... }, ... }
tupper by id
GET /api/tuppers/{tupperID}
Example: localhost:8080/api/tuppers/1
Update tupper
POST /api/tuppers/{tupperID}
Example: POST http://localhost:8080/api/tuppers/1 Header: Content-Type: application/json Body: { "description": "new Description", "location", { city: "new City", ... } }
